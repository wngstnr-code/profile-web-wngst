MERN Authentication System

Sistem autentikasi full-stack berbasis MERN (MongoDB, Express, React, Node.js) dengan fitur lengkap untuk manajemen pengguna modern. Mendukung registrasi dan login dengan validasi email, OTP verifikasi, reset password via email, serta proteksi rute menggunakan JWT. Dibangun dengan antarmuka responsif dan pengalaman pengguna yang intuitif.

Teknologi :

Frontend: React 18, Vite, React Router, Axios, Tailwind CSS

Backend: Node.js, Express.js, MongoDB (Mongoose), JWT, Bcryptjs, Nodemailer
